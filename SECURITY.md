# Security Policy

Open Agentic Threat Defense is defensive security software. Reports about
vulnerabilities in this project are welcome.

## Reporting

Send reports to: security@example.invalid

Include:

- affected version or commit;
- impact;
- reproduction steps;
- suggested mitigation if known.

## Scope

In scope:

- vulnerabilities in the server, API, policy engine, dashboard, or test assets;
- unsafe default behavior;
- response actions that could exceed dry-run or configured permissions.

Out of scope:

- requests for offensive features;
- exploit development;
- live-target testing without authorization.
